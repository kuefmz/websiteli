# Websiteli Marketing Automation System

This folder contains the operating system for Websiteli's 90-day content, SEO, outreach, demo, and reporting workflow.

## Core principle
Use one central orchestrator and reusable templates. Do not create many disconnected agents.

## Safety rule
Cold outreach must be prepared as review-ready drafts. Do not auto-send cold emails or DMs.

## Connected systems
- GitHub: source control for content and demo assets
- Google Sheets: CRM, content calendar, SEO log, outreach log, weekly review
- Gmail: review-ready outreach drafts
- Google Drive: working docs and tracker storage

## Daily workflow
1. Generate one content package.
2. Generate SEO metadata and indexing checklist.
3. Research 5 qualified leads.
4. Draft personalized outreach only for approved leads.
5. Select 1 top lead for a demo concept when useful.
6. Update the tracker.

## Weekly workflow
1. Review published content performance.
2. Review leads, replies, and pipeline movement.
3. Pick best countries/niches for next week.
4. Improve prompt templates based on replies and conversions.
